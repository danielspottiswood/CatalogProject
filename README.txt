Daniel Spottiswood Catalog Project

Purpose: This project allows users to post, edit, and delete items in a few specific categories: cleaning, entertainment, and tools.

Setup:

1. Clone and download the project.
2. Navigate into the catalog directory
3. Set up the database with the following command: "python database_setup.py"
4. Populate the database with the following command: "python database_pop.py"
5. Run the web application with the following command: "python application.py"
6. Access the application in your browser at the following URL "http://0.0.0.0:8000/"
